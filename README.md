# No Hello

A single-page guide to async communication etiquette, inspired by [nohello.com](https://nohello.com) and [nohello.co](https://nohello.co).

## What's covered

| Section | Etiquette |
|---------|-----------|
| **No Hello** | Don't just say "hi" and wait — include your question in the first message |
| **No Call** | Don't call unannounced — send a message instead; calls should be scheduled or urgent |
| **No Connect** | Don't send blank connection requests — add a short personal note explaining why you're reaching out |

## Usage

Open `index.html` directly in any browser — no build step or dependencies required.
